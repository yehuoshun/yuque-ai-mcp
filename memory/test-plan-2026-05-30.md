### 3. 测量指标

收集每个 query 的：
- **命中率**：索引文档是否被搜到（是/否）
- **覆盖率**：预期源文档中实际被搜到的比例
- **权重排序**：返回结果是否按 w 降序排列（检查 ⭐ 标记）
- **降级触发**：是否有 0 命中需要二轮补搜

### 4. 异常场景

| 场景 | 预期行为 |
|------|---------|
| 搜索 token 在总库 0 命中 | 返回「未找到匹配的索引域，尝试降级全库搜索」 |
| 子库搜到索引但 0 entries | 返回「未找到相关内容，请尝试换个问法」 |
| 同 did 出现在多个关键词索引 | did 去重，保留 w 较高的那次 |
| 超长代码文档 | 截断后仍可提取关键词，不报错 |
