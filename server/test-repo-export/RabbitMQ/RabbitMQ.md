## <font style="color:rgb(36, 41, 46);">课程简介</font>
<font style="color:rgb(102, 102, 102);">RabbitMQ是实现了高级消息队列协议（AMQP）的开源消息代理软件（亦称面向消息的中间件）。本课程内容包括环境搭建、后台监控管理界面、发送队列、交换机、三种绑定模式、使用Java代码实现最基本的接受消息和发送消息。高级特性包括主题匹配、削峰机制、死信队列、在SpringBoot中整合RabbitMQ实现快速首发消息。本课程由浅入深，让你可以循序渐进的理解RabbitMQ的工作原理和开发步骤。</font>
